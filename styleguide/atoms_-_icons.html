Icon font

Using IcoMoon, we've created an updateable icon font to use across the site.

Usage is as simple as applying the 'icon' class and the 'icon-[NAME]' class to a span (or anchor wrapping a span) to choose the icon used. If you need the larger size, also add the 'icon--large' class.

These font-sizes (32px default, 54px large) don't change over breakpoints, as discussed with Dan Ridge.

See the examples below, check out the wiki documentation for full documentation here under 'Icons':
http://wiki.comicrelief.com/index.php?title=RND15TechnicalDocumentation#SASS_Components
