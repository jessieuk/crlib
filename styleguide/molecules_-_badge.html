Usage

Add the 'badge' and 'badge--VARIANT' classes to a relativelty positioned element to receive a nifty tag, made with pure CSS. Because of IE8's issues with :afters, :befores and z-indexing, these are hidden on that particular browser.

This element will also need to be free of any other :before or :after pseudo elements in order to work.

Type Modifier Class
'.badge .badge--video'
'.badge .badge--event'
'.badge .badge--photo'
'.badge .badge--news'
