Rows

The following classes for the row component all respond to all breakpoints.

The row styles are created in HTML by adding the required row class modifiers to an <div> or a <section> with a .row-- class. There is no core .row class as that name is reserved for Bootstrap is is not used here.

See the examples below.

Row base class

The class .row--content includes all default margins and paddings accross all breakpoints.

Classes to use Example of use in HTML
.row--content <div class="row-content">

</div>

How to use Rows across the site

Specific Row classes are only required when they are specifically mentioned below, if not the default row class (added already) will work fine.

Row Small Copy

Body copy below should be <h3 class="type--centred type--white type--margin-large">Some text</h3>

Row Small copy



Row Medium Copy

Row class should be pane-text--medium-copy while the title should be <h2 class="type-centred">Title</h2> and the copy should be <p class="type--lg type--centred"></p>

Row Medium Copy


Row Large Copy

The title should be <h2 class="type-centred type--white type--row-large-copy">Title</h2> while the first paragraph should be <p class="type--lg type--lg-negative-margin type--centred--xs type--entred--sm type--white"></p> and the second paragraph should be <p class="type--white type--centred--xs type--entred--sm"></p> and the button should be <p class="type--centred--xs type--centred--sm type--row-large-margin-top"><a class="btn btn-color btn-ghost"></a></p>

Row Large Copy


Row Main Center

Row class should be pane-image--row-center while the title should be <h3 class="type--centred type--margin-large"></h3> and the button should be <p class="type--centred "><a class="btn btn-color"></a></p>

Row Main Center


Row Quote

The quote should be <h2 class="type--centred type--margin-none"></h2>

Row Quote


Row Two Messages

Row class should be pane-image__wrapper--multiple-images
Header should be <h2 class="type--centred"></h2>
Two sets of mini panels are required, the image in the first mini panel should have class pane-image--first-class applied
Mini panels should be set to 12xs 6sm and 4md with a 2md offset for the first one
Title should be <h3 class="type--centred"></h3>
Copy should be <p class="type--centred"></p>
Button should be <p class="type--centred"><a class="btn btn-color"></a></p>
