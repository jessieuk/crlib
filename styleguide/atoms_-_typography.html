HR

If you need a simple coloured line, you can render a <hr> tag with one of the .hr--{colour} classes, as shown below.

Note: the <hr> doesn't get any margin or padding so you'll need to supply this in your own component, should you need to.

Sizes

There are 10 different type sizes, each with their own variants. Apply the class in the modifier to the size and style you would like. If the modifier simply say e.g h4 then no class is needed.

There are a list of classes related to brand colours, use .type--white, .type--red for example.

Special Circumstances

These have been displayed below on p tags but they can be attached to any tag, p tags are just used for demonstration. You should find everything you need here, if not, let me know.

Text Aligning

There are classes below to show text being aligned, there are also breakpoint specific classes such as; .type--left--sm, .type--right--sm, .type--centred--sm. Note, these classes do not inherit upwards in a mobile first fashion, they work between breakpoints. The example will work for the SM breakpoint only.
